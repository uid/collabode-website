Collabode
Collaborative Coding in the Browser

Collaborating programmers should use a development environment designed specifically for collaboration, not the same one designed for solo programmers with a few collaborative processes and tools tacked on.


Collabode is a web-based Java integrated development environment built to support close, synchronous collaboration between programmers.


Collabode uses Eclipse on the server-side to manage projects and power standard IDE services: continuous compilation, compiler errors and warnings, code formatting and refactoring, and execution.

On the client, Collabode uses EtherPad to support collaboration between multiple simultaneous editors. Any number of programmers can open the same file simultaneously, and their concurrent changes are shared in near real-time to enable smooth collaboration.


The Collabode research project aims to demonstrate how a programming environment built to support specific structures of close collaboration can improve both the quality of the collaboration and the software produced.
